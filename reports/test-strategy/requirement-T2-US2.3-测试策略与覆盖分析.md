## 11. 风险和前置条件

| 项目 | 说明 | 影响 | 建议 |
|---|---|---|---|
| 后端 API 未覆盖 | 8 个 API 案例均未覆盖 | 接口契约、权限、参数校验风险 | 优先执行 L2 API 自动化 |
