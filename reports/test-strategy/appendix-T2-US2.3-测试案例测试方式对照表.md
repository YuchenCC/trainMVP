## 4. 不建议放进单测的内容

| 内容 | 原因 |
|---|---|
| HTTP 请求/响应格式验证 | 需要 API 层集成测试，单测 mock 无法验证真实契约 |
| 数据库实际写入和读取 | 需要 API 层或集成测试验证数据库副作用 |
| 前端页面渲染和交互 | 需要 L3 E2E 测试，单测无法验证 DOM 渲染 |
| 跨模块事务一致性 | 需要集成测试验证多表写入和事务回滚 |
