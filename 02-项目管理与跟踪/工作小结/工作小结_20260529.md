### 2.2 Coze SDK Bug 修复

| 问题 | 根因 | 修复 |
|------|------|------|
| 审查 `Missing parameter: title` | `parameters: { parameters: {...} }` 双重嵌套 | 改为 `parameters: parameters`（SDK 自动包装） |
| 前端 10s 超时 | 全局 axios timeout 太小 | 改为 180s（3分钟） |
| 验收条件不展示 | 前端状态类型缺少 `acceptanceCriteria` | 补充类型 + Tabs 渲染 |
