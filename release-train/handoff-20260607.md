# Handoff Document

## Purpose
修复导览功能退出后重复启动的问题。

## Completed Work
- **问题诊断**：识别出 React StrictMode 导致 useEffect 触发多次，导致导览退出后重复启动
- **解决方案**：在所有页面添加 `hasProcessedTour` ref 防止重复触发
- **测试验证**：通过 Playwright 自动化测试，确认退出导览后不再重复启动

## Test File

## Key Decisions
- 使用 `useRef` 而非其他方式防止重复触发，因为 ref 不参与组件重渲染
- 在 `skipTour()` 中清除 URL 参数，避免页面状态变化导致重复处理
- 保持调试日志一段时间，方便后续排查问题
