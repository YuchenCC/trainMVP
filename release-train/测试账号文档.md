## 三、功能入口地址

> **前提**：确保已启动前后端服务
> ```bash
> cd release-train/apps/server && pnpm dev   # 后端 → http://localhost:3000
> cd release-train/apps/web && pnpm dev      # 前端 → http://localhost:5173
> ```
> 未登录状态下访问任何页面会自动跳转登录页，建议用 `puhuiqianduan_ba`（张大伟）登录体验完整功能。

| 功能 | 前端地址 | 后端接口 | 说明 |
|------|----------|----------|------|
| 登录页 | [login](http://localhost:5173/login) | `POST /api/auth/login` | 输入账号密码登录 |
| 需求池首页 | [requirements](http://localhost:5173/requirements) | — | 需求列表 +「新增需求」入口 |
| 新增需求 | [requirements](http://localhost:5173/requirements) → 点「新增需求」 | `POST /api/requirements` | 双栏布局（左表单+右实时预览） |
| 版本火车 | [trains](http://localhost:5173/trains) | — | 版本火车列表（待开发） |
