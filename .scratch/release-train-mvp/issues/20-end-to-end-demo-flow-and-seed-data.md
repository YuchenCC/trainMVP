# 端到端演示流程与种子数据

Status: ready-for-agent
Type: AFK

## What to build

准备覆盖 MVP 关键场景的演示数据和端到端演示流程，让系统可稳定展示需求录入到纳版、已纳版变更回退、AI建议确认纳版、投产、回滚和日历跳转。

## Acceptance criteria

- [ ] 种子数据覆盖多角色、多系统、多需求状态和多趟版本火车。
- [ ] 种子数据覆盖容量预警、容量超限、依赖未满足、依赖已取消、依赖系统未参车。
- [ ] 端到端流程可演示从需求创建到评审、纳版、投产和回滚。
- [ ] 端到端流程可演示已纳版变更回退和重新评审。
- [ ] 端到端流程可演示 AI建议确认纳版和日历跳转详情。
