# 测试方案与验收用例设计

Status: ready-for-human
Type: HITL

## What to build

围绕 PRD 验收标准设计测试方案，明确单元测试、集成测试、页面流程测试和演示验收脚本的覆盖范围，优先验证外部可观察行为。

## Acceptance criteria

- [ ] 覆盖需求状态机、容量、依赖风险、纳版、权限、AI建议、审计和页面流程。
- [ ] 每条 PRD MVP 验收标准都有对应验证方式。
- [ ] 区分自动化测试、人工验收和演示脚本。
- [ ] 明确种子数据需要覆盖的状态、风险和角色场景。
