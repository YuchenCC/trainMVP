# 需求池基础闭环

Status: ready-for-agent
Type: AFK

## What to build

实现需求池的基础端到端能力，让业务 BA 和产品经理可以创建、编辑、查询需求，并完成草稿、待评审、已就绪、已拒绝和已取消等基础状态流转。

## Acceptance criteria

- [ ] 支持需求创建、编辑、列表查询和详情查询。
- [ ] 新建需求默认进入草稿状态。
- [ ] 支持发起评审、评审通过、评审拒绝、拒绝后重新编辑。
- [ ] 支持投产前需求取消，取消原因必填并生成操作审计。
