# 版本火车创建与搭载系统配置

Status: ready-for-agent
Type: AFK

## What to build

实现版本火车创建和搭载系统配置能力。火车创建员可以创建版本火车，配置参与系统、系统成员和团队容量，系统自动计算结束时间与统一纳版、统一封板、统一投产节点。

## Acceptance criteria

- [ ] 支持创建版本火车并设置开始时间、周期和描述。
- [ ] 自动计算结束时间和关键节点。
- [ ] 支持配置一个或多个搭载系统及本车角色、团队容量。
- [ ] 同一系统参与火车时执行闭区间周期重叠校验。
