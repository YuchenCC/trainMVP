# 火车详情与容量快照

Status: ready-for-agent
Type: AFK

## What to build

实现版本火车详情页和容量视图，按搭载系统展示本期可用点数、已分配点数、剩余可用点数、容量使用率，以及 90% 预警和超限风险。

## Acceptance criteria

- [ ] 火车详情展示基础信息、状态、关键节点和搭载系统。
- [ ] 按系统计算已分配点数、剩余点数和容量使用率。
- [ ] 容量使用率达到 90% 时提示预警。
- [ ] 容量超限时提示软风险，不直接阻断后续强制纳版。
