# 完成版本火车

Status: ready-for-agent
Type: AFK

## What to build

实现版本火车手动完成能力。系统在统一投产节点后，且本车不存在仍处于已纳版状态的需求时，提示火车管理员可以完成版本火车，由管理员手动确认完成。

## Acceptance criteria

- [ ] 版本火车支持计划中、进行中、已完成、已取消状态。
- [ ] 系统能判断并提示版本火车是否满足完成条件。
- [ ] 火车管理员可手动确认完成版本火车。
- [ ] MVP 不依赖定时任务自动完成版本火车。
