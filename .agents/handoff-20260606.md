### 飞书消息卡片结构
```json

  "card": {
    "header": {"title": "📦 代码已提交推送", "template": "blue"},

      {"tag": "div", "text": {...}},  // 项目、分支、提交信息

      {"tag": "div", "text": {...}},  // 变更文件列表
```
